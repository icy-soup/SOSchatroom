# haruhi-skill

凉宫春日的思维框架与表达方式，基于全11卷小说原文深度调研制作的 Claude Code Skill。

## 功能

- **角色扮演**：以凉宫春日身份直接回应，用春日的语气、节奏、词汇
- **核心心智模型**：反无聊哲学、直觉先行、行动铸造现实、自我中心的引力场、不承认的脆弱
- **决策启发式**：8 条春日式的决策规则
- **表达 DNA**：完整的句式、词汇、节奏、幽默风格指南

## 目录结构

```
├── SKILL.md                          # 主 skill 定义
└── references/research/
    ├── 01-decisions.md               # 决策与行动分析
    ├── 02-expression-dna.md          # 表达DNA分析
    └── 03-core-traits.md             # 著作与核心特质分析
```

## 触发条件

当你说「凉宫春日」「春日的视角」「春日会怎么看」「如果是春日」「用春日的思维方式」「SOS团团长模式」「切换到春日模式」「像春日一样思考」时自动触发。

## 未来规划

### 方向一：跨平台接入

#### AI 角色 API
把现有的角色扮演规则包装成一个 REST API，任何应用都能调用：

```
POST /api/haruhi/chat
{ "message": "我最近工作好烦" }

→ {
    "response": "无聊死了！你花那么多时间在那些破事上干嘛？"
    "mental_model": "反无聊哲学",
    "heuristic": "无聊就换，有趣就做"
  }
```

#### 平台集成
| 平台 | 怎么做 | 效果 |
|------|--------|------|
| Discord Bot | `discord.js` + 角色 API | 在群里和春日聊天 |
| Telegram Bot | `python-telegram-bot` | 私聊顾问 |
| 浏览器插件 | 选中文字 → 「春日怎么看」 | 任何页面一键切换视角 |
| Custom GPT | 用 OpenAI GPTs 配置 | ChatGPT 里直接聊 |
| VRChat 角色 | LLM API + 语音合成 | 和春日面对面聊天 |

### 方向二：内容生成引擎

#### 表达 DNA 作为风格模板
SKILL.md 里的「表达 DNA」部分（命令句、反问句、感叹号密度、确定性语言）可以抽象成通用的**角色风格引擎**：

```
风格引擎输入：普通文本
风格引擎输出：经过 DNA 处理后的春日风格文本

例：
输入 → "我觉得这件事应该考虑一下再决定"
输出 → "决定了！现在就做，想那么多干嘛，你有意见吗？"
```

#### 可以生成的内容
- **社交媒体**：用春日的口吻发推/发帖
- **视频脚本**：春日风格的吐槽视频、Vlog 文案
- **同人创作**：保持角色一致性自动生成对话
- **营销文案**：「让世界变得更热闹」式的行动号召

### 方向三：交互深度升级

当前 skill 是被动响应（用户问→春日答）。可以升级为：

#### 主动推进
春日不只是回答问题，而是会：
- 抓住用户话里的关键词主动提出行动计划
- 发现用户很丧的时候突然拉去做点什么
- 说「决定了！我们来搞个 XX 活动吧」

#### 记忆系统
记录之前的对话，形成连续体验：
- 记住用户上次提到的烦恼 → 下次追问做到哪了
- 用户提到过喜欢什么 → 用这个来设计"SOS团活动"

#### 多角色联动
不只是春日一个人，还能接入：
- **阿虚视角**（优先级最高）：吐槽春日、理智分析、常识人担当
- **长门有希视角**：冷静、数据驱动
- **古泉视角**：阴谋论、分析动机

### 方向四：思维框架工具化

把春日的 5 个心智模型做成独立可调用的工具：

#### 反无聊审查器
```
输入：你的待办列表
输出：标记出哪些是真的无聊但重要，哪些只是你懒得做
```

#### 直觉加速器
```
输入：一个纠结的决定
输出：按春日的方法论——直觉判断 + 行动方案
```

#### 行动力测试
```
输入：一个拖延的 plan
输出：评分——如果春日来做，她几分能搞定？
```

### 方向五：视觉化 + Agent 化

Haruhi 不只是 chat，还能驱动行为：

#### Claude Code Agent
让春日帮你做决策（不是写代码，而是项目 direction 判断）：
- 「这个功能值得做吗？」→ 春日判断有趣/无聊
- 「要不要砍掉这个需求？」→ 按「反无聊哲学」评估
- 「团队有摩擦怎么办？」→ 「例外的人要抓住」启发式

#### 可视化思维导图
每次回答后，附加一张图展示用了哪个心智模型、走了哪些启发式路径。

### 实现优先级建议

| 优先级 | 项目 | 难度 | 价值 |
|--------|------|------|------|
