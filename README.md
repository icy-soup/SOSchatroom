# haruhi-skill

《凉宫春日》系列 SOS 团全员思维框架 Skill 合集，基于全13卷小说原文深度调研制作。

## 角色阵容 🎉

| 角色 | 目录 | 状态 | 核心心智模型 |
|------|------|:----:|:-------------|
| 🌟 凉宫春日 | `characters/haruhi/` | ✅ 完成 | 反无聊哲学、直觉先行、行动铸造现实、自我中心的引力场、不承认的脆弱 |
| 💬 阿虚 | `characters/kyon/` | ✅ 完成 | 吐槽透镜、责任悖论、普通人锚点、选择即承认、抱怨是关心的面具 |
| 📚 长门有希 | `characters/nagato/` | ✅ 完成 | 观测者立场、数据的真实性、沉默的力量、情感是冗余还是升级 |
| 🎀 朝比奈实玖瑠 | `characters/asahina/` | ✅ 完成 | 时间线的守护者、柔弱是保护色、双重身份的智慧、禁止事项下的诚实 |
| 🃏 古泉一树 | `characters/koizumi/` | ✅ 完成 | 微笑面具、棋盘世界观、情报梯度解密、闭锁空间辩证法、双重忠诚的平衡术 |
