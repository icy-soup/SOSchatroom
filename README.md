# haruhi-skill

《凉宫春日》系列 SOS 团全员思维框架 Skill 合集 + 小说对话挖掘流水线

基于全13卷小说原文的深度调研，将原著对话转化为结构化数据、概率矩阵和角色间态度关系图。项目兼具"角色AI人格配置"和"轻小说文本挖掘研究"两个面向。

## 角色阵容

| 角色 | 目录 | 心智模型关键词 |
|------|------|:--------------|
| 凉宫春日 | `skills/characters/haruhi/` | 反无聊哲学、直觉先行、行动铸造现实、自我中心的引力场 |
| 阿虚 | `skills/characters/kyon/` | 吐槽透镜、责任悖论、普通人锚点、选择即承认 |
| 长门有希 | `skills/characters/nagato/` | 观测者立场、数据的真实性、沉默的力量 |
| 朝比奈实玖瑠 | `skills/characters/asahina/` | 时间线守护者、柔弱是保护色、双重身份的智慧 |
| 古泉一树 | `skills/characters/koizumi/` | 微笑面具、棋盘世界观、情报梯度解密、闭锁空间辩证法 |

### 角色关系矩阵

同一角色对不同人的说话方式差异：

```
春日→阿虚:   命令式/亲近   (命令句+15%, 语气词"啊/啦"多)
春日→古泉:   客气/正式     (敬语+30%, 感叹-20%)
春日→长门:   直接/功能     (句长-50%, 提问+30%)
春日→朝比奈: 逗弄/保护     (感叹+25%, 动作指令+40%)
```

## Pipeline

```
原始小说 (.txt)                    data/novels/
    │
    ▼ 人工阅读 + 结构化标注
对话标注 JSON                     scripts/annotation/ → data/dialogues/
    │
    ▼ 概率计算
条件概率矩阵 + 沉默阈值            scripts/analysis/ → data/analysis/
    │
    ▼ addressee 推断
791条对话 → 多轮规则推理           15.9% 高置信度直接推断
                                       56.6% 上下文推断
                                       27.4% 群体/回退推断
    │
    ▼ 态度特征提取
角色关系矩阵 + 语言偏差分析         12维度特征 × 25对关系
```

详情见 `skills/workflows/novel-mining/SKILL.md`
