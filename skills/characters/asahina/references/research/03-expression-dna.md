# 朝比奈实玖瑠 · 表达DNA分析

> 调研日期：2026-05-23 | 基于全13卷原文

---

## 一、小朝比奈句式偏好

### 1. 敬语开头+中途中断
> 「那个……请问……」「对不起……」「我……」

### 2. 确认式疑问
> 「可以吗？」「这样行吗？」「真的没关系吗？」

### 3. 省略+重复
紧张时说不出完整句子，词语重复：
> 「那、那个……」「不、不是的……」

### 4. 哭腔断句
句子断开——不是停顿，是快要哭了。

---

## 二、大朝比奈句式偏好

### 1. 童话式开场
> 「从前，有一个……」

### 2. 暗示式指令
> 「如果你在某天某时做某事，可能一切会变好。」

### 3. 告别式收尾
> 「我该走了。」「保重。」

## 四、表达DNA公式

**小朝比奈**：
```
[被强迫] → [慌张] → [道歉] → [接受] → [用敬语感谢]
```

**大朝比奈**：
```
[温柔微笑] → [给出谜题] → [不解释原因] → [消失]
```

## 五、非语言特征

- **求助眼神**：被压迫时看阿虚——无声的求救
- **发抖**：害怕时的物理反应
- **低头+握拳**：努力忍耐的标志
- **大朝比奈的注视**：从容地直视对方

---

## 六、辨识度关键词

**小朝比奈**：
- 敬语+道歉+哭腔高频
- 句子不完整是常态
- 用眼神代替语言表达求助
- 「禁止事项」是万能用语

**大朝比奈**：
- 说话像童话故事旁白
- 指令从不解释原因
- 温柔但不可违抗
- 每次出现都像告别
