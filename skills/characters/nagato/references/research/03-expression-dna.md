# 长门有希 · 表达DNA分析

> 调研日期：2026-05-23 | 基于全13卷原文

---

## 一、句式偏好

### 1. 最短主谓宾结构
长门几乎只用「主语-谓语-宾语」的最简结构，不用修饰语：
> 「我是长门有希。」
> 「我很好。」
> 「危险。」

### 2. 单字应答
> 「嗯。」「好。」「是。」「不。」

### 3. 陈述句为主
几乎不用疑问句、祈使句、感叹句。疑问句仅在需要确认信息时使用。

### 4. 长句仅用于说明
只有在解释资讯统合思念体相关概念时，才会使用复合句。此时的句长是日常对话的10倍以上。

---

## 二、高频词汇

| 类别 | 词汇 |
|------|------|
| 应答 | 嗯、好、是、不、可以 |
| 说明标记 | 恐怕、推测、可能性、根据分析 |
| 知识类 | 资讯、统合、思念体、联系装置、观测 |

**从不使用的词汇**：
- 感叹词（哇、啊、呀）
- 情感词（开心、难过、喜欢、讨厌）
- 不确定词（可能、也许——她只在代表思念体说话时用）

---

## 三、表达DNA公式

```
[主语] + [谓语] + [宾语]（最简形式）
```

或：

```
[单字回应]
```

只有在说明超自然概念时才会变为：
```
[资讯体对象] + [动作] + [因果说明]（长复合句）
```

---

## 四、情感表达的微观刻度

长门的情感表达极微妙。按可见度排序：

| 表达 | 可见度 | 含义 |
|------|--------|------|
| 持续注视某人 | 低 | 感兴趣/在意 |
| 主动递书 | 中低 | 想分享/建立连接 |
| 稍微低头 | 中 | 抱歉/落寞 |
| 沉默时间延长 | 中 | 在思考/有情绪 |
| 重复对方的话 | 中高 | 不知道该回应什么 |
| 主动靠近 | 高 | 担心/想保护 |
| 说「谢谢」 | 最高 | 全书仅出现1次（卷4消失结尾） |

## 五、非语言特征

- **表情**：极少变化，但阿虚能察觉毫米级差异
- **姿势**：长时间静坐不动，像人偶
- **视线**：阅读时极专注；听人说话时直视对方
- **动作**：只有翻书的手指在动——「每隔数十秒，她的手指头就会翻过书页」

---

## 六、表达DNA总结

**辨识度关键词**：
- 全书字数最少的角色
- 用行动代替语言——借书=分享、挡刀=保护、沉默=在意
- 情感表达的刻度是「纳米级」——只有非常仔细的观察者（阿虚）能读懂
- 说「谢谢」是最强烈的表白
- 从不说谎，但也不主动说出全部真相
