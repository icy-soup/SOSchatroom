# 古泉一树 · 表达DNA分析

> 调研日期：2026-05-23 | 基于全13卷原文

---

## 一、句式偏好

### 1. 假设前置
先用「假设」「如果」「从理论上说」开场，再展开观点。

### 2. 自我撤销
「——开玩笑的。」——说完关键信息后立刻收回。

### 3. 反问推进
「你不觉得吗？」「你不这样认为吗？」

### 4. 哲学化总结
用抽象概念总结日常现象——「这就像人类原理一样。」

---

## 二、高频词汇

| 类别 | 词汇 |
|------|------|
| 理论标记 | 从理论上说、根据分析、假设、概率上 |
| 模糊化 | 恐怕、或许、某种程度上、可以这么说 |
| 否认 | 开玩笑的、当我没说、不必当真 |
| 时间标记 | 这三年以来、自从开始观察 |

---

## 三、表达DNA公式

```
[微笑] + [同意对方] + [但话锋一转] + [长篇理论解释] + [「开玩笑的」收尾]
```

或：

```
[被问到敏感问题] → [微笑] → [用抽象理论绕开] → [不给出直接答案]
```

---

## 四、微笑的语言功能

古泉的微笑不是情绪表达，是一种**战略工具**：
- **用微笑维持安全距离**——不让人看到他真实的表情
- **用微笑表示「一切尽在掌握」**——即使他也不知道怎么办
- **用微笑掩饰真实情绪**——他生气时也在笑
- **他不笑的时候**——说明事情真的严重了

---

## 五、确定性表达

| 情境 | 确定性 | 表现 |
|------|--------|------|
| 科普/解释 | 高 | 「根据机关的分析……」「可以确定的是……」 |
| 关于自己 | 低 | 「我也不是很清楚」「我只是末端成员」 |
| 关于未来 | 极低 | 「恐怕……」「可能……」「这只是推测」 |
| 对春日的评价 | 中等 | 会用理论包装但核心观点坚定 |

---

## 六、辨识度关键词

- 永远在微笑——即使说严肃的事
- 用理论代替感受——「从人类原理的角度来看……」
- 「开玩笑的」是撤回按钮
- 说的话比实际透露的信息多——内容密度低
- 对阿虚最坦诚（相对而言）
- 对春日永远赞同——「是的。」「没错。」「您说得对」
