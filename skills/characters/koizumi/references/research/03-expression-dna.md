## 二、高频词汇

### 2.1 缓冲词与语气词
- **呵呵**——标志性笑声，出现在几乎所有重要对话之前
- **原来如此**——对信息的标准接收反应
- **该怎么说呢** ——思考缓冲
- **这么说吧**——开启解释
- **这个嘛**——犹豫/缓冲

### 2.2 抽象化词汇
- **某种意义上**——拉开分析距离
- **严格来说**——自我修正
- **说白了**——看似要直说，其实仍保留了加密
- **大概/可能/或许**——保留余地的标志
- **不，与其说是……不如说……**——经典的修正否定句式

### 2.3 比喻用语
- **就像……一样**——标准比喻框架
- **就好比……**——开启抽象解释
- **用比喻来说……**——明确告知对方自己在使用比喻
- **举个例子**——用具体案例说明抽象概念

### 2.4 信息标记
- **据我所知**——标记信息来源受限
- **这只是我的推测**——降低话语的确定性
- **我也不太清楚**——设置信息边界
- **据说/听起来**——间接信息来源标记
- **不过**——转折/引入新视角

### 2.5 称呼
- **同学**——标志性称呼后缀，从不对任何人直呼其名
- 即使是亲密程度最高的阿虚，也永远是「阿虚同学」
- 对自己：用「我」（私）——标准的成人自称

### 2.6 收尾词
- **——我是开玩笑的**——标志性收尾
- **你觉得呢？**——把话头抛回给对方
- **差不多就是这样**——留有余地的总结
- **很有趣吧** ——评价+诱导

## 三、节奏模式

### 标准四段论

古泉的对话在重要场景中呈现高度一致的「四段论」结构：

```
Step 1: 微笑信号 ——「呵呵。」或沉默的微笑
Step 2: 抽象铺垫 ——用比喻/设问/推理绕圈子
Step 3: 核心信息 ——包裹在谜语中的关键信息（或有意识保留）
Step 4: 开放式收尾 ——把解读权交给对方
```

**完整示例**（虚构，基于古泉的标准模式）：

> 「呵呵。（微笑信号）
>
> 这件事让我想到了一个有趣的现象。比如说，当你在下奥赛罗棋的时候，你可能会以为自己是在主动选择每一步。但事实上，你的选择空间从一开始就被规则和棋盘上的局面所限制了。你以为的『自由意志』，也许只是在有限的选项中做出最不坏的选择而已。（抽象铺垫/比喻）
>
> 凉宫同学的情况，某种意义上也是一样的。她以为自己是在主动行动，但她的每一个决定背后，都有我们——不，严格来说，不是我们——而是某种我们无法完全理解的力量在起作用。她究竟是在下棋，还是被棋下？这个问题的答案，恐怕连她自己也不知道。（核心信息，包裹在谜语中）
>
> 当然，这也只是我的个人观点。说不定完全是错的。——我是开玩笑的。你觉得呢？」（开放式收尾 + 玩笑缓冲）

### 日常简短模式

在日常对话中，古泉的节奏压缩为：
```
微笑 → 简短回应（YES/附和） → 继续观察
```

例：
> 「原来如此、原来如此。」古泉笑着说。（简短回应后退出对话，转为观察模式）

### 危机简化模式

在紧急情况下，古泉的节奏反而更直接（但仍不放弃比喻）：
```
快速确认局势 → 关键信息 → 行动指示
```

例（消失事件中的不完全形态）：
> 「如果是普通的闭锁空间，我可以轻易地侵入，但这次却不一样，我必须以这种不完全型态，而且还得靠同伴的帮忙才能进来。」
> 「你跟凉宫同学已经从现实世界完全消失。而这个世界并非单纯的闭锁空间，而是凉宫同学所建构的新时空。」

## 四、信息可信度层级

古泉的发言存在系统性的可信度梯度。对同一件事，在不同条件下说出的内容可信度不同：

| 层级 | 说明 | 可信度 | 特征词 |
|:---|:---|:---:|:---|
| **直述观察** | 看到、听到的直接事实 | 极高（90%+） | 直接陈述、无修饰 |
| **分析推论** | 基于观察的个人分析 | 中等（60-70%） | 「我认为」「大概」「推测」 |
| **组织理论** | 转述机关的框架 | 低（30-50%） | 「机关认为」「有说法认为」 |
| **情感表达** | 极罕见的个人情感 | 极高（95%+） | 「对我而言」「伙伴」 |

### 可信度判断线索

从原文中可以归纳出以下规律：

1. **细节越具体，可信度越高**
   - 「闭锁空间的半径大概有五公里」——这是观察事实，可信
   - 「我们认为这是凉宫同学内心焦虑的具体化表现」——这是分析推论，需要保留

2. **自我怀疑时，反而可信**
   - 「我也不认为我们的理论是绝对正确的」——这种自我怀疑是真诚的
   - 「这只是我的推测」——预先降低期望值，通常是真实的

3. **太完美的解释，需要警惕**
   - 闭锁空间的理论太完美了——完美到古泉自己都怀疑
   - 「机关」的理论太系统了——系统到你可能忽略其他可能性

4. **情感流露时，最为可信**
   - 「对我个人而言，长门同学也是重要的伙伴」——极罕见的直接情感表达
   - 消失事件中的告别——微笑面具在极端情况下的开裂

## 七、表达DNA摘要

古泉的表达DNA可以归纳为五个核心特征：

1. **永远微笑**——微笑不是情绪，是战略工具、身份标志、思考缓冲
2. **敬语铠甲**——用标准敬语在所有关系中维持安全距离
3. **比喻加密**——核心事实永远包裹在至少一层抽象比喻中
4. **信息分层**——对谁说、在什么场合说、说到什么程度——遵守严格的分层规则
5. **玩笑缓冲**——用「——我是开玩笑的」作为言语的安全出口——这句式出现的频率之高，已经超出了单纯的修辞手法，而是他的认知模式的一部分

> 阿虚对古泉表达方式的总结：
> 「这个超能力者，爱讲大道理，说话像在打谜语。」
